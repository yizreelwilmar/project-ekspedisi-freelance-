// Requirement
1. XAMPP
2. PHP (version 8.0.25, not tested with other version yet)

// HOW TO INSTALL
1. Copy and extract this folder to xampp/htdocs
2. Create database with the name db_pengiriman in phpmyadmin
3. Import database file from /db/db_pengiriman.sql
3. Run this system in a browser, then input login information
4. Enjoy !

// FEATURES
1. Dashboard - Summary of production progress & history
2. Package - Set volume paket, customer information, product order and volume of Package
3. Product - Product data, includes weight and volume product
4. Cargo - Process load capacity by adding route points and package volume, including truck data and driver data
5. Truck - Set truck data, max capacity and max volume
6. Driver - Add driver data
7. Shipping - Processing delivery of cargo data, and processing package delivery status by driver
8. Reports - Delivery Reports of shipping status
